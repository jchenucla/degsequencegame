How to Play

Objective: Draw a simple graph based on the degree sequence shown on the screen before the timer runs out!

1. Click on the node that you wish to connect.

2. Click on another node that you wish to connect the node to.

3. Repeat steps 1 and 2 to create the graph before the timer runs out!

4. Use the "Check" button to verify if the graph matches the degree sequence.
